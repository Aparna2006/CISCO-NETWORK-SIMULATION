# CISCO-NETWORK-SIMULATION

## Cisco Packet Tracer Projects by Aparnaa Kondiparthy

### About this repository
This repository is a curated collection of Cisco Packet Tracer labs and simulations created by **Aparnaa Kondiparthy** for practical networking learning and certification preparation. It covers essential Cisco routing, switching, and security topics through hands-on Packet Tracer scenarios.

### What is included
- **Basic Projects**
  - Default Routing Implementation
  - DHCP Server Configuration
  - FTP File Server Configuration
  - Static Configurations on Routers
- **CCNP Labs**
  - Access Control Lists (ACL)
  - Overlay Tunnels (VPN) with GRE
- **Intrusion Detection System**
  - Complete IDS lab with Packet Tracer project file and report

### Project goals
- Demonstrate real-world network design and troubleshooting
- Build practical Cisco Packet Tracer labs with working topology and configuration
- Reinforce CCNA/CCNP concepts such as routing, DHCP, ACL, VPN overlay, and security monitoring

### Usage
1. Install Cisco Packet Tracer.
2. Open the `.pkt` project files in the corresponding folders.
3. Review each folder’s README and lab documentation before starting.
4. Use the provided topology and configurations as a study guide or practice lab.
5. Do not overwrite original files; save copies if you want to experiment.

### Skills demonstrated
- Static routing and default routing configuration
- DHCP server setup and IP addressing
- FTP server deployment and client connectivity
- Router and switch configuration for basic network operation
- Standard and extended ACL creation and enforcement
- GRE tunnel setup for overlay VPN-style connectivity
- Intrusion detection fundamentals and security analysis

### Important / mandatory notes
- These labs are designed for **Cisco Packet Tracer**. Open the `.pkt` files with the correct Packet Tracer version.
- Read the folder-specific README files in each lab before making changes.
- Keep a backup of original files before editing or saving changes.
- If a lab uses private address ranges, do not expect the topology to route on the public internet.
- Use the contained lab diagrams and commands as a reference for study and practice.

### Why this project matters
This repository is a practical portfolio of networking labs created by **Aparnaa Kondiparthy**. It is intended for students and professionals who want a structured, project-based way to learn Cisco network configuration, security, and troubleshooting within Packet Tracer.
